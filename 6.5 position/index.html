CSS POSITION PROPERTY

The CSS position property specifies how an HTML element is positioned on a web page. It controls the placement of elements using the properties top, right, bottom, and left.

position: static

It is the default position of all HTML elements. Elements follow the normal document flow. The properties top, left, right, and bottom do not work. Example div { position: static; }

position: relative

The element is positioned relative to its normal position. Space for the element is still reserved. It is commonly used as a parent reference for absolutely positioned elements.

EG:.box { position: relative; top: 20px; left: 30px; }

position: absolute

The element is positioned relative to the nearest positioned ancestor (not static). If no such ancestor exists, it is positioned relative to the body. The element is removed from normal flow. No space is reserved for it.

EG:.parent { position: relative; } .child { position: absolute; top: 10px; right: 10px; }

position: fixed

The element is positioned relative to the browser window (viewport). It does not move while scrolling. Commonly used for navigation bars, chat icons, and ads.

EG:.navbar { position: fixed; top: 0; left: 0; }

position: sticky

The element behaves like relative until a scroll position is reached. After that, it behaves like fixed. It sticks to the specified position while scrolling.

EG:.header { position: sticky; top: 0; }

Additional Properties

top, right, bottom, left Used to move positioned elements. Works only with relative, absolute, fixed, and sticky.

EG:top: 10px; left: 20px;
